# 机器人使用示例

myagv 的典型用例场景通常涉及与机械臂配对组成复合机器人，从而实现移动抓取、扩大工作空间并完成更广泛的任务。以下是 myagv 支持的机械臂。

**myCobot 280 M5/Pi**

**myPalletizer 260 M5/Pi**

**mechArm 270 M5/Pi**

**myarm 300 pi**

**ultraArm P340**
