## 1 myStudio 设计初衷

- myStudio 是使用机器人的一站式平台。

- 用户可以根据自己的使用场景选择不同的固件并下载，同时还可以在线学习相关教材和浏览教程视频，非常方便。

## 2 myStudio 最新版本和支持的平台

- 最新版本： V3.5.7

- 可用平台

  - Windows
  - Mac
  - Linux arm64

## 3 myStudio 功能

- 刻录和更新固件。
- 提供机器人使用教程，如用户手册、视频教程、问答等。
- 保养和维修信息。
